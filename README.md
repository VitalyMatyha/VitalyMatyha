## 🚀 Проекты

### 📋 [Employee Task Tracker](https://github.com/VitalyMatyha/employee-task-tracker) 
> REST API для управления задачами сотрудников

- Django 6 + Django REST Framework + PostgreSQL + Docker
- Кастомные API-вью с аннотациями ORM (подсчёт задач, фильтрация по статусу)
- Покрытие тестами **99%**, соответствие PEP8 / flake8
- Автодокументация через **drf-spectacular** (Swagger / ReDoc)

---

### 🧠 [Habit Tracker](https://github.com/VitalyMatyha/habit-tracker) — трекер полезных привычек
> Приложение на основе книги «Атомные привычки»

- Кастомная модель пользователя + JWT-аутентификация
- Асинхронные задачи на **Celery + Redis** — напоминания через **Telegram-бота**
- CI/CD через **GitHub Actions**, деплой на **Yandex Cloud** (Docker Compose + Nginx)
- Публичный API для обмена привычками между пользователями

---

### 🎓 [LMS Platform](https://github.com/VitalyMatyha/lms-platform) — платформа онлайн-обучения
> Система управления курсами и уроками

- Курсы, уроки, платежи: интеграция с **Stripe API**
- Разграничение прав: модераторы и владельцы контента
- Периодические задачи Celery (блокировка неактивных пользователей)
- Swagger-документация через **drf-spectacular**, деплой на **Yandex Cloud**

### 📊 [HH Vacancy Analyzer](https://github.com/VitalyMatyha/hh-vacancy-analyzer) — анализ вакансий
> Парсинг и аналитика данных с HH.ru

- Получение данных о компаниях и вакансиях через **HH.ru API**

### 💰 [Finance Dashboard](https://github.com/VitalyMatyha/finance-dashboard) — финансовая статистика
